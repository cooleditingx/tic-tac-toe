### Installation

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/cooleditingx/tic-tac-toe.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd tic-tac-toe
    ```
Open the index.html file in your web browser.

***Note on Assets:*** The game requires three image files to be present in the root directory for full visual fidelity:
- `background.png`: For the page background.
- `arrow.png`: Represents the 'X' mark.
- `circle.png`: Represents the 'O' mark.

## How to Play

1.  The game begins with "Player 1 Turn". Player 1 uses the 'X' mark.
2.  Click on an empty square on the board to place your mark.
3.  The turn then switches to Player 2, who uses the 'O' mark.
4.  Players alternate turns until one player achieves three of their marks in a row, column, or diagonal.
5.  The game ends when a player wins or when all nine squares are filled, resulting in a draw.
6.  The game status is always visible above the board, indicating the current player's turn or the final outcome.
